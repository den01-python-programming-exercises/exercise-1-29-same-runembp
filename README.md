# Exercise 1.29 - Same

Write a program that prompts the user for two strings. If the strings are the same, then the program prints "Same". Otherwise, it prints "Different".

```plaintext
Enter the first string:
*hello*
Enter the second string:
*hello*
Same
```

```plaintext
Enter the first string:
*hello*
Enter the second string:
*world*
Different
```
